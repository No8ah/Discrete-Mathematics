$$\tag{谓词}用于\underline{\textbf{刻化个体属性}}或者\underline{\textbf{表达个体之间关系}}的词语$$
## 记号 :
$$用\underline{\textbf{大写字母}}表示$$
## 例子 ：
### 谓词 :
#### 例1 ：
$$S \ : \ \cdots\ 是大学生$$
$$a \ : \ 小张$$
$$b \ : \ 小李$$
$$命题 \ : \ 小张(小李)是大学生$$
$$表示 \ : \ S(a) \ 、 \ S(b)$$
#### 例2 ：
$$Q \ : \ \cdots > \cdots$$
$$命题 \ : \ 3  > 7$$
$$表示 \ : \ Q \ ( \ 3 \ ,\ 7 \ )$$
#### 例3 ：
$$B \ : \ \cdots \ 在 \cdots \ 与 \cdots \ 之间 $$
$$命题\ : \ 点a  \ 在 \ 点b \ 与 \ 点c \ 之间$$
$$表示 \ : \ B \ ( \ a \ , \ b \ , \ c \ )$$
